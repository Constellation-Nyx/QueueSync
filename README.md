# QueueSync

Real-time hospital queue management system built using React, Node.js, Express, and Socket.IO.

## Features
- Real-time queue updates
- Receptionist dashboard
- Patient dashboard
- Token-based tracking
- Estimated wait times
- Activity logs

## Tech Stack
- React
- Vite
- Node.js
- Express
- Socket.IO

## Run Locally

Frontend:

npm install
npm run dev

Backend:

npm install
node server.js
